# Polynomial processing
This is a GUI `Java` application that works with polynomal processing and operations.  
### Supported  operations:
-  addition
-  substraction
-  multiplication
-  division
-  derivation
-  integration
-  works with all kind of inputs such as `5x+5+5+3` or `44x^0+44+44x+44x^1` etc.
